TOPIC 2

What is the difference between Inline and Block tags?

Some tags are inline. Such tages are br for 'break' and a for anchor and img for an image. These tags are out in space and no matter the spaccing things will just follow one after the other. with no layout or formating specified.

Other tags are block tags. Such tags are div, and p for paragraph. These create an invisible box around the text or image and they specify some layout attributes.

TOPIC 3

How do Tags work in HTML?

Some tags are void --- that means that they do not need to be closed. and the 'br' tag is like this. Some tags have attributes. Attributes give more information. Like the anchor tag carries the attribute 'href' which points us to the link. Also, the img tag carries the attribute 'src' which points us to the image file.

TOPIC 6

What is the difference between class and id?

ID gives us a way to uniquely identify an element in an entire webpage. It gives a way to style just that one element. It should only refer to one element. CLASS is used to refer to a group of similar elements in a webpage.
